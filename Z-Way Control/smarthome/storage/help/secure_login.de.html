Ich vertraue meinem lokalen Ethernet/WLAN

WLAN Verbindungen werden verschlüsselt und eine kabelgebundene Kommunikation im Haus ist sehr sicher gegen Lauschangriffe. Wenn Sie trotzdem Zweifel haben, das ihr lokales Netz sicher ist dann wird jede Verbindung zu ihrem Controller über unseren sicheren Server mit gültigen Verschlüsselungs-Zertifikaten umgeleitet. Dies kann jedoch zu Verzögerungen und einer geringeren Bandbreite führen.

Hinweis: Sichere IP-Verbindungen erfordern gültige Serverzertifikate. Da ihrem Controller jedoch eine dynamische IP-Adresse zugewiesen wurde, kann kein Zertifikat vorinstalliert werden. Wir verzichten daher bei direkter Kommunikation im lokalen Netzwerk auf die Verschlüsselung. Sichere komplett verschlüsselte Verbindungen werden dann über unseren Server umgeleitet, auf dem gültige Zertifikate installiert wurden
